# Contributing

Thanks for helping improve OpenPasture. This project is focused on the Morning
Farm Brief experience and adaptive grazing intelligence.

## Development Setup

Prerequisites: Node.js 18+ and npm.

```bash
cd app
npm install
npm run dev
```

## Development Workflow

- Keep changes small and focused.
- Run `npm run lint` before opening a pull request.
- If you touch the UI, include screenshots or short clips when possible.

## Pull Requests

- Use a descriptive title and summary.
- Explain the why, not just the what.
- Include a brief test plan (even if it is "not tested").
- Update documentation when behavior changes.

## Issues

Please use the issue templates for bug reports and feature requests to help us
triage quickly.

## Code of Conduct

By participating, you agree to follow the [Code of Conduct](CODE_OF_CONDUCT.md).
